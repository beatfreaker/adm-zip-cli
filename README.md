# adm-zip-cli
Extract .zip file from command line

## Usage
```
    > unzip --help

      Usage
        > unzip <filename> <folderpath>
        
        <filename>   - file you waant to extract
        <folderpath> - to which folder you want to extract the files
    
      Examples
        > unzip filename.zip foldername
        
```
